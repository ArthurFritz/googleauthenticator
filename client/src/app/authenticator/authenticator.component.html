Uma Abordagem da Utilização Do Google Authenticator para aumentar segurança do software

O Objetivo deste artigo é propor uma aplicação de baixo custo, para ter um aumento de segurança do software, trazendo consigo a aplicação do Google Authenticator.

O protótipo utilizará a linguagem de programação Java juntamente com o Framework Spring Boot, e para seu back-end incorporado consigo para o front-end a tecnologia Angular, abordando uma simples utilização da validação de tokens gerados pelo Google Authenticator.
